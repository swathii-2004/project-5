# ProxiMart: Real-Time Local Product Discovery & Reservation Platform

## Overview

ProxiMart is a smart hyperlocal platform that enables users to check real-time product availability in nearby stores and reserve items for instant pickup — bridging the gap between offline retail and digital convenience, without the delays of traditional e-commerce.

---

## Problem Statement

Local shoppers often face stock uncertainty, wasted travel time, and no visibility into what nearby stores actually have in stock. Meanwhile, local businesses lack the digital tools to compete with large e-commerce players. ProxiMart solves both sides of this problem by connecting users to real-time local inventory and giving vendors a powerful digital storefront.

---

## Core Roles

### 👤 User
- Search products within a selected radius
- View real-time availability and compare stores by price, distance, and ratings
- Reserve items with a **"book now, pay at store"** model
- Receive smart notifications for stock updates and price drops
- Navigate to stores with pickup route guidance

### 🏪 Vendor
- Apply to join and manage a digital storefront upon admin approval
- Add products and sync inventory in real time
- Confirm reservations and access demand analytics
- View insights on popular products and peak booking times

### 🛡️ Admin
- Approve and verify vendors
- Monitor platform activity and ensure security
- Manage users, listings, and overall ecosystem health

## Key Features

| Feature | Description |
|---|---|
| 🔍 Smart Local Search | Filters like "available now", "best price nearby", distance-based results |
| 📦 Real-Time Inventory Sync | Accurate, live stock availability across all stores |
| 🗺️ Live Map View | Nearby stores displayed with product availability overlays |
| ⚡ Quick Reservation System | Time-limited booking with no upfront payment required |
| 🏬 Multi-Store Comparison | Compare price, distance, and ratings before reserving |
| 🤖 AI-Based Recommendations | Personalized suggestions based on user behavior and local trends |
| 📊 Vendor Analytics Dashboard | Demand forecasting, popular products, and peak time insights |
| 💬 In-App Chat with Vendors | Pre-booking queries directly within the platform |
| 🔔 Smart Notifications | Alerts for stock updates, price drops, and reservation status |
| 🚨 Emergency Mode | Quickly locate essential or urgent items nearby |
| 👥 Group Reservation | Bulk or shared reservations for group needs |
| 📶 Offline Lite Mode | Displays last-known availability in low-network conditions |
| ✅ Trust & Verification System | Admin-approved vendors with ratings and reviews |
| 🔐 Role-Based Access Control | Secure, separate portals for Admin, User, and Vendor |

## Tech Scope *(Proposed)*

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Node.js / FastAPI
- **Database:** MongoDB / PostgreSQL
- **Maps:** Google Maps API / Mapbox
- **Auth:** JWT with role-based access
- **Notifications:** Firebase Cloud Messaging
- **Payments:** Razorpay / Stripe *(at store, optional digital)*
